notchtap · Status Rail — showcase rev 2 (proposed changes)

To answer the variant question directly: there is one card component, not multiple variants. Everything on this page is that single card in different states (idle / compact / expanded) and skins (3 priority accents, 6 news categories, per-signal stamps). What changed in this rev: rounded corners by default (Round 16px), tier chip (L1/Low) deleted — priority now reads from the accent edge + stamp + track color only, track repurposed as a queue slider (one segment per queued item, advancing automatically), auto-expand lifecycle for every item (expand on show → retract → rotate on), and five celebration candidates to judge. Opacity: yes, three shipped presets — Glass 0.7 / Default 0.9 / Solid 1.0 — in the bar below; Default stays selected.

0 · Full lifecycle — auto-expand, retract, queue slider

The proposed flow, end to end: item promotes already expanded (regardless of priority) → holds → retracts to compact → holds → rotates to the next queued item, or drains to idle when the queue is empty. The bottom track is the queue slider: 5 segments for 5 queued items — dim = consumed, bright = current. Hold durations here are stand-ins (2.2s expanded / 1.6s compact) since the real timer is already configurable.

1 · Celebration candidates — pick one (or a combo)

A is the shipped goal celebration rechecked: 620ms scale overshoot + radial burst + confetti (confetti reworked — bigger particles, fly-out then gravity fall; still a CSS stand-in for the Lottie comp). 2 · Priority skins — color only, chip deleted

The L1/M2/H3 chip is gone. Priority now reads from: the 3px accent edge on the left, the stamp, and the queue track's lit color. Same card, three accents.

3 · Expanded state (500px, manifest)

Width morph 320ms + manifest height/opacity 240ms. Toggle to judge. In the proposed flow this state plays automatically on every promotion (section 0); the manual ⌃⇧N toggle still exists on top.

4 · News cards (RSS) — all six categories

Unchanged apart from the chip removal and queue track. Masthead + 13px headline + pills + the 12s shade-drift tint.
