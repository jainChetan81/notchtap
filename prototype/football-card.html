Football card — sticky presence, event-driven expansion

Sticky medium-priority resident from ~30 min before kickoff to full-time: resting = idle rail with the green dot, hover reveals the compact scorecard, no full-expand. What forces the card open is a high-level event — goals, penalties, own goals, yellow and red cards get celebrations; plan 043's richer feed (fouls, offsides, VAR checks, substitutions, via item 6a's summary→plays fallback) expands the card quietly. Then it settles back to the rail. Paused football → hover falls back to weather; higher-priority items preempt, football returns.

Event icon set below is the item-6 proposal — CSS shapes, no emoji: goal (filled ball), penalty (ring), own goal (hollow), yellow/red (slips), foul (triangle), offside (flag), VAR (monitor), substitution (up/down). Goal/yellow/red tint their event line and celebrate; 043's informational events stay neutral.
